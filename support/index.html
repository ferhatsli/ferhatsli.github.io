Support URL Setup

This page serves as the official support URL for your iOS app in App Store Connect. You can reference this URL in your App Store listing using:

https://ferhatsli.github.io/support

How to Use This Support URL

1. In App Store Connect

  • Log in to App Store Connect
  • Select your app
  • Go to "App Information"
  • In the "Support" section, enter: https://ferhatsli.github.io/support
  • Save your changes

2. In Your App

  • Add a support button or menu item in your app's settings
  • When users tap the support option, open this URL in their default browser
  • Example Swift code: 
    if let url = URL(string: "https://ferhatsli.github.io/support"),
   UIApplication.shared.canOpenURL(url) {
    UIApplication.shared.open(url)
}
